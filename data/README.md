# Data Directory

This directory is reserved for approved POC assets.

Recommended structure:

- `data/templates/`
  - built-in PPTX templates, such as t5 and t6
- `data/samples/`
  - approved sample source documents, such as the Baosteel annual report

Do not store runtime logs, generated PPTX outputs, PNG previews, temporary uploads,
cache files, or customer-uploaded templates here.
